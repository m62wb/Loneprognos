# Lönekalkylator IKEM - Uppsala
Detta är en webbaserad lönekalkylator för
IKEM:s lokala avtal, byggd för skiftarbetare i Uppsala med skattetabell 33.
**Funktioner:**
- Stöd för fem olika skiftlag (A-E) med
automatiska scheman
- Automatisk OB-beräkning (storhelg, helg, vardag)
- Semestertillägg, VAB/Föräldraledighet,
Karens, Sjuklön
- Skatt enligt tabell 33 (Uppsala)
- Fackavgift (IF Metall)
- Spara/ladda scenarier i webbläsaren
- Mörkt/ljust tema
**Filstruktur:**
- 'index.html' - huvudstruktur och CSS
- 'js/skattetabell.js' - skattetabell 33
- 'js/scheman.js" - schema för alla lag,
storhelger, OB-beräkning
- 'js/arsoversikt.js' - årsöversikt och
reset av OB
- 'js/script.js' - huvudlogik, beräkningar och UI
- 'js/storage.js' - scenariohanterare (localStorage)
